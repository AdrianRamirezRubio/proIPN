Volumenes de Revolución

Cuando giramos la región de área que está debajo de una función alrededor del eje x O algún otro eje determinado, lo que estamos obteniendo es un sólido. El volumen de este solido también se obtiene con la integral.

Esto porque siempre vamos a girar una superficie y tendremos siempre como elementos de sección pequeños cilindros de área πr2

Para calcular el volumen, es el mismo procedimiento del cálculo de áreas, solo que ahora la función se eleva al cuadrado. Y se tiene que hacer la gráfica y además hacer una figura tomando en cuenta el giro de la función.

Hay que realizar la gráfica y tener una mente abierta, para poder determinar cuando gira la función sobre el eje de las x o de las x como se vería la figura.
